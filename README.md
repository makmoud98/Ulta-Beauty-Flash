# Ulta Beauty Flash
Using NLP, Machine Learning, and Voice to enhance and simplify the ulta customer experience


## What it does
- Recommend items to new and existing users (Using Collaborative and Content Filtering Algorithms)
- Find nearest ulta stores
- Find availability of products in ulta stores
- An overall personalized voice experience

## How we built it
- Python (Spacy, Numpy, Pandas, Scikit-learn, Flask)
- Google Cloud
- Google Assistant


## Running the environment
pip3 install -r requirements.txt
